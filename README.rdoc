It is inspired by the original [timesheet plugin](https://github.com/edavis10/redmine-timesheet-plugin), written by Eric Davis.

## Redmine Compatible
The plugin is tested on Redmine 4.2

## Features

Just a link of issues#Show

## Installation and Setup

1.  Clone this repository to your Redmine plugins directory:

    `$ git clone https://github.com/i4dev/redmine_top_issues`

2. Execute migrations:

     `RAILS_ENV=production bundle exec rake redmine:plugins:migrate`

3.  Restart web server
